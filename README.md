# bxsite

The fastest and easiest way to build a website. You can have a free site just like this in minutes.

## How it works

1. Create an account
2. Write your site content in markdown
3. Choose your domain name
4. Click deploy

## Get started

Go to the [dashboard](/dashboard) and create your first site.
